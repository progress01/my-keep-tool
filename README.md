# my-keep-tool(參考至防彈筆記)
一個單頁式的網頁工具，專門用來標準化與生成「防彈筆記」格式。設計核心是為了配合 Google Keep 的使用習慣，透過結構化的區塊來管理專案進度與任務。
## 主要功能
* **智慧解析**：支援貼上既有的筆記內容，系統會自動辨識標題並將內容填入對應的編輯欄位。
* **結構化管理**：將筆記強制分為狀態更新、資源庫、目標/風險、ABCD 執行清單與關鍵字。
* **連結整理**：提供工具按鈕，可將雜亂的網址自動斷行排列。
* **一鍵輸出**：生成符合特定排版格式的純文字，方便貼回 Google Keep 或其他筆記軟體。

## 欄位說明
工具將筆記分為以下區塊：
1.  **狀態更新 (Status)**：紀錄目前的進度或下一步驟。
2.  **專案資源庫 (Links)**：存放 SOP、雲端硬碟或其他參考連結。
3.  **目標與風險**：確保任務核心目標明確，並標註潛在風險。
4.  **執行清單 (ABCD)**：
    * A. 暖身準備 (Ready)
    * B. 核心任務 (Main)
    * C. 成果交付 (Delivery)
    * D. 經驗存檔 (Review)
5.  **設定檢查**：提醒標註急迫性（紅/綠）與提醒事項（時間/地點）。

## 使用方式
1.  **開啟工具**
    直接使用瀏覽器（如 Chrome, Edge）開啟 `index.html` 檔案。
2.  **讀取舊檔（可選）**
    若要更新既有筆記，將原本的筆記內容複製，貼入右側面板的「讀取舊進度」框，並點擊「解析並填入左側」。
3.  **編輯內容**
    在左側面板對各個欄位進行編輯或新增任務。
4.  **輸出筆記**
    編輯完成後，查看右側面板的「最終結果」，點擊「全選並複製」按鈕，將內容貼回筆記軟體中。

## 技術資訊
* **格式**：單一 HTML 檔案（包含 CSS 與 JavaScript）。
* **相容性**：支援現代主流瀏覽器。
* **資料隱私**：所有運作皆在瀏覽器端本機執行，不會上傳資料至任何伺服器。
